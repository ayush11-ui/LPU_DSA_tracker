# 2. Min Stack (Medium)

### Problem
Design a stack that supports the following operations in **O(1)** time:
- `push()`
- `pop()`
- `top()`
- `getMin()`


### Python Solution

```python
class MinStack(object):

    def __init__(self):
        self.s=[]
        self.m=[]

    def push(self, value):
        self.s.append(value)
        self.m.append(value if not self.m else min(value,self.m[-1]))

    def pop(self):
        self.s.pop()
        self.m.pop()

    def top(self):
        return self.s[-1]

    def getMin(self):
        return self.m[-1]
```
