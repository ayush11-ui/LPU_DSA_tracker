# 3Sum

## Problem Statement

Given an integer array `nums`, return all the unique triplets `[nums[i], nums[j], nums[k]]` such that:

- `i`, `j`, and `k` are distinct.
- `nums[i] + nums[j] + nums[k] == 0`.

The solution set must not contain duplicate triplets.

---

## Python Solution

```python
class Solution:
    def threeSum(self, nums):
        nums.sort();r=[]
        for i,a in enumerate(nums):
            if i and a==nums[i-1]:continue
            l,h=i+1,len(nums)-1
            while l<h:
                s=a+nums[l]+nums[h]
                if s<0:l+=1
                elif s>0:h-=1
                else:
                    r+=[[a,nums[l],nums[h]]];l+=1
                    while l<h and nums[l]==nums[l-1]:l+=1
        return r
```

---

## Example

**Input**

```text
nums = [-1,0,1,2,-1,-4]
```

**Output**

```text
[[-1,-1,2],[-1,0,1]]
```
