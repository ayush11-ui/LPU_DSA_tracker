# Two Sum II - Input Array Is Sorted

## Problem Statement

Given a sorted integer array `numbers` and an integer `target`, return the **1-indexed** positions of the two numbers such that they add up to `target`.

There is exactly one solution, and you may not use the same element twice.

---

## Python Solution

```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        l,r=0,len(numbers)-1
        while numbers[l]+numbers[r]!=target:
            if numbers[l]+numbers[r]<target:l+=1
            else:r-=1
        return [l+1,r+1]
```

---

## Example

**Input**

```text
numbers = [1,2,3,4]
target = 3
```

**Output**

```text
[1,2]
```
