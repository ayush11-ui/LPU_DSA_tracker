# Container With Most Water

## Problem Statement

You are given an integer array `height` where `height[i]` represents the height of the `iᵗʰ` bar.

Choose any two bars to form a container and return the maximum amount of water the container can store.

---

## Python Solution

```python
class Solution:
    def maxArea(self, h):
        l,r=0,len(h)-1
        ans=0
        while l<r:
            ans=max(ans,min(h[l],h[r])*(r-l))
            if h[l]<h[r]:
                l+=1
            else:
                r-=1
        return ans
```

---

## Example

**Input**

```text
height = [1,7,2,5,4,7,3,6]
```

**Output**

```text
36
```
