# Merge Sorted Array

## Question 1: Merge Sorted Array (LeetCode 88)

### Problem Statement

You are given two sorted arrays:

* `nums1` has size `m + n`.
* The first `m` elements are valid.
* The last `n` positions are empty (`0`s are placeholders).
* `nums2` contains `n` sorted elements.

Merge `nums2` into `nums1` so that `nums1` remains sorted.

### Example

```python
nums1 = [1,2,3,0,0,0]
m = 3

nums2 = [2,5,6]
n = 3
```

Output:

```python
[1,2,2,3,5,6]
```

---

## Important Terms

### `m`

Number of valid elements in `nums1`.

```python
nums1 = [1,2,3,0,0,0]
         ↑ ↑ ↑
```

Only these elements are considered.

### `n`

Number of elements in `nums2`.

### In-place

Modify `nums1` directly without returning a new array.

---

## Easy Solution

```python
class Solution(object):
    def merge(self, nums1, m, nums2, n):
        nums1[m:] = nums2
        nums1.sort()
```

## Optimal Solution

```python
class Solution(object):
    def merge(self, nums1, m, nums2, n):
        i, j, k = m-1, n-1, m+n-1

        while i >= 0 and j >= 0:
            if nums1[i] > nums2[j]:
                nums1[k] = nums1[i]
                i -= 1
            else:
                nums1[k] = nums2[j]
                j -= 1
            k -= 1

        while j >= 0:
            nums1[k] = nums2[j]
            j -= 1
            k -= 1
```

### Complexity

* Time: **O(m+n)**
* Space: **O(1)**

# Question 2: Merge Two Sorted Arrays Without Extra Space

### Problem Statement

Merge two sorted arrays without creating a third array.

Example:

```python
arr1 = [1,4,7,8,10]
arr2 = [2,3,9]
```

Output:

```python
arr1 = [1,2,3,4,7]
arr2 = [8,9,10]
```

---


### Without Extra Space

Do not create another array like:

```python
arr3 = arr1 + arr2
```

Only modify the given arrays.

---

## Easy Solution

```python
def merge(arr1, arr2):
    for i in range(len(arr2)):
        if arr1[-1] > arr2[i]:
            arr1[-1], arr2[i] = arr2[i], arr1[-1]
            arr1.sort()
    arr2.sort()
```

---

## Optimal Solution (Gap Method)

### Idea

Use a gap between compared elements (similar to Shell Sort). Reduce the gap until it becomes 1, swapping out-of-order elements.
